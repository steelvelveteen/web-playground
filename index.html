data attributes enables you to store extra information that doesn't have any visual representation. data-*

Javascript access

To read them in javascript use getAttribute() or via the dataset object

'const article = document.querySelector('#electric-cars')'

and then article.dataset.columns will give the stored value

You can set the data value from the javascript as above

Css access

As attributes they are they can be accessed from CSS

        article:before {
            content: attr(data-parent);
        }
